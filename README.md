- View details of meal

## Running locally

Clone the project

```bash
  git clone https://github.com/felipefreitasa/dailydiet.git
```

Enter the project directory

```bash
  cd dailydiet
```

Install the dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```
